# 知识库 B 站学习助手（Chrome / Edge）

这是本项目的本地浏览器扩展。它让 B 站在真实页面中按你的账号权限播放，并在右侧边栏显示当前章节笔记。

## 安装

1. 双击项目根目录的 `启动知识库.cmd`，保持知识库已打开。
2. 在 Chrome 或 Edge 地址栏打开扩展管理页：`chrome://extensions` 或 `edge://extensions`。
3. 开启右上角「开发者模式」。
4. 点击「加载已解压的扩展程序」，选择本 `browser-extension` 文件夹。
5. 将扩展固定到工具栏。

## 使用

在知识库的课程章节中点击「B站学习」。扩展会打开对应 B 站视频，并尝试自动展开右侧边栏。若浏览器没有自动展开，点击工具栏的扩展图标即可。

如果章节媒体区显示「高清悬浮学习」，点击它后扩展会打开 B 站官方页面。先确认账号登录和清晰度，再点击右下角出现的「进入画中画并返回笔记」一次；浏览器会把真实 B 站视频放入系统悬浮窗，并自动切回知识库。这一步必须由你点击，浏览器不允许扩展静默开启画中画。

点击侧栏「保存进度」会保存下次从哪里继续观看，不影响正文。点击「插入单点」会把当前时间写入章节正文。若要记录片段，先在开始处点「设为起点」，填写旁注，播放到结束处后点「插入片段」。正文会自动加入类似 `🎬 19:00–21:00｜这里讲解 xx 软件的安装过程` 的条目；插入片段也会把续播进度保存到片段末尾。

扩展不会读取或保存 B 站 Cookie，也不上传笔记或视频数据。

## 已知范围

- 仅支持 Windows/macOS/Linux 桌面版 Chrome 或 Edge。
- 仅处理完整的 `www.bilibili.com/video/BV...` 链接。
- B 站的登录、清晰度、会员和地区规则完全由 B 站页面决定。
- 画中画窗口使用浏览器系统控件；需要改画质、弹幕或倍速时，请回到对应 B 站页面操作。
