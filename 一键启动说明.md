# 一键启动、关闭与分享

完整功能说明见 [README.md](README.md)。

## 本机使用

1. 双击 `启动知识库.cmd`。
2. 脚本会构建最新前端、启动本地服务，并自动打开 `http://127.0.0.1:4173`。
3. 结束使用时双击 `停止知识库.cmd`。

首次使用需要 Node.js LTS；之后不需要输入任何终端命令。

## 分享给其他人

- `dist/`
- `scripts/local-server.mjs`
- `启动知识库.cmd`、`停止知识库.cmd`
- 可选：`media/`（需要附带本地视频时）

每位使用者的数据都在自己的浏览器本地，不会共享、同步或上传。

## 发布新版

修改源码或从 GitHub 拉取新版后，再次双击 `启动知识库.cmd`。网页仍显示旧版时，按一次 `Ctrl + Shift + R`；不要清除站点数据，以免误删本地笔记。
