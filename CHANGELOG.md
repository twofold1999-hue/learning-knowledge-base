# 更新日志

本项目采用 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 的简要格式；版本号遵循语义化版本的发布意图。

## [0.1.0] - 2026-07-22

首个可在本地运行和演示的候选发布版本。

### 新增

- 本地优先的 Markdown 笔记、课程与有序章节管理；
- 标题和正文全文搜索、Wiki 链接、前向/反向链接及孤立笔记浏览；
- 年度笔记创建足迹、学习进度与随机回顾；
- 笔记图谱，以及仅展示已确认实体和关系的只读实体图谱；
- 实体详情、关联笔记、证据关系与知识变更审计浏览；
- AI 笔记整理、知识候选审核、来源追踪与审计记录；
- Backup v5 JSON 导出、验证、合并恢复与本地目录备份；
- 隔离演示数据、产品截图、生产 E2E 与手工验收基础设施。

### 改进

- 收窄长正文编辑时的渲染边界，优化保存生命周期和 Wiki 链接查询；
- 引入轻量笔记投影，避免完整正文在笔记列表中长期常驻；
- 优化实体图谱的加载、布局、工具栏响应式布局和首屏可读性；
- 完善年度创建足迹的日期语义、键盘访问、Tooltip 与日期笔记导航；
- 将编辑器概览、链接、AI 历史和知识辅助内容集中到可折叠辅助面板；
- 强化生产 E2E 稳定性，并完成性能基线和独立 Microsoft Edge 手工验收。

### 质量

- 已记录性能基线，并完成隔离演示数据与真实 Edge 独立 Profile 验收；
- 已知限制和适用边界见 [README](README.md#已知限制) 与 [v0.1.0 发布说明](docs/releases/v0.1.0.md)。
- 328 项 Vitest 测试通过；
- 26 项常规 Node 契约测试通过；
- 3 项发布版本契约测试通过；
- 33 项生产 Playwright E2E 通过。

### 安全

- 将 DOMPurify 更新至 3.4.12，修复候选版本依赖审计发现的 HTML 清理器安全问题；
- 更新受影响的开发工具传递依赖，消除候选发布审计中的高危告警。
